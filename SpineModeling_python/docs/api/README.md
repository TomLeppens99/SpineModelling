# API Documentation

This directory contains detailed API documentation for the SpineModeling Python package.

## Modules

- **core**: Core data models and structures
- **imaging**: Medical image processing and analysis
- **visualization**: 3D rendering and visualization
- **algorithms**: Core algorithms for biomechanical analysis
- **ui**: User interface components
- **database**: Database layer and persistence
- **utils**: Utility functions and helpers

## Generating Documentation

To generate API documentation using Sphinx:

```bash
cd docs/
sphinx-build -b html . _build
```

The generated documentation will be available in `docs/_build/index.html`.
